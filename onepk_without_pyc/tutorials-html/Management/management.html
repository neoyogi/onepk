Overview

The Application CLI Extension Service Set enables onePK applications to add features to a network element's operating system and to store their specific configurations on the network element.

These tutorials demonstrate how to configure the connected network element and how to use the Application CLI Extension Service Set functionalities.

Application CLI Extension Configuration This tutorial demonstrates the configuration needed for the Application CLI Extension Tutorial.
Application CLI Extension Tutorial This tutorial shows how the Application CLI Extension service enables a onePK application to add commands to the network element's command-line-based management interface (CLI), which can then be used to configure the application from the network element.
