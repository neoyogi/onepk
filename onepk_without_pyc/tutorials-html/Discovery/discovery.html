Overview

The Discovery Service Set provides two mechanisms for onePK applications: service discovery and topology discovery.

  • Service discovery gets information about onePK services running on a local connected network element.
  • Topology discovery gets information about the network topology as seen by a protocol.

These tutorials demonstrate the Discovery Service Set functionality.

Basic Discovery Service Sets Tutorial This tutorial gets and displays the list of service sets available on the connected network element.
Topology Overview & Tutorial Overview about the topology service and tutorial that registers a CDP listener and looks for neighbors on the connected network element.
