Overview

These tutorials are used to demonstrate the Network Interface Service Set Capabilities.

Basic Network Interface Tutorial This tutorial shows how to access Interface(ports/links), their properties and events from onePK-enabled Network Elements.
Interface Address Tutorial This tutorial gets and displays Interface Addresses and Prefixes, and sets Interface Addresses.
Interface State Change Tutorial This tutorial creates and registers an InterfaceStateListener and then causes events to be generated by changing the status of Network Interfaces.
Interface Statistics Tutorial This tutorial creates and registers an InterfaceStatisticsListener, and also shows how to poll for Interface Statistics.
Interface CDP Tutorial This tutorial creates and registers an InterfaceCDPListener
